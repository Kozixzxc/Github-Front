# Github-Front

반응형 깃허브 페이지 클론디자인입니다.

설명
-----------
- opengraph를 설정하여 외부 검색엔진에 정보 제공

- Webkit ms moz를 사용하여 주로 사용하는 브라우저에서 모두 정상적으로 보일 수 있도록 Placeholder 디자인

- header, section , wrapper , inner로 구조를 나눠서 코딩

- display를 block처리하여 float속성을 해제하는 기능을 가진 class clearfix를 구현하여 class로 지정(캡슐화, 재사용)

- 유투브에서 제공하는 영상 비율은 16:9를 유지하고 있으므로 padding을 맞춰 선언해주고 Iframe을 선언해 영상이 비율에 맞춰 확대 축소

- 4컬럼 이미지를 반응형으로 제작하기 위해 flex 대신 grid 컨테이너를 사용하여 제작

- 지도 API 삽입


Editor
------------

- Visual Studio Code


사용 언어 , 프레임워크 , 폰트
-----------

- html,css(vanila)

- google font
